컴파일
javac -d mods/me.hjyoon src/me.hjyoon.multichat/module-info.java src/me.hjyoon.multichat/me/hjyoon/multichat/Main.java

한 번에 모두 컴파일
javac -d mods --module-source-path src $(find src -name "*.java")
또는
javac -d mods --module-source-path src --module me.hjyoon.multichat

실행
java -p mods -m me.hjyoon.multichat/me.hjyoon.multichat.Main

※ --module-path는 -p로 줄여 쓸 수 있습니다.
※ --module-source-path는 실제 모듈의 소스코드가 위치한 루트 디렉터리를 가리킨다.
※ --module는 -m로 줄여 쓸 수 있습니다. 그리고 -m 옵션은 main 모듈을 지정한다.
